Introduction

Project Overview

Planet Granite is an interactive portfolio built around a space exploration theme. Each planet represents a stage of the development journey.

Design Concept

The design is inspired by orbital motion and exploration. The user travels through stages of development as if moving through space.

Technical Approach

The site is built using HTML, CSS, SVG path animation, and JavaScript for interactive motion and background rendering.
